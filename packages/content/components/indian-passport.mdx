## Overview

The Indian Passport Input component provides a specialized input field with built-in validation for Indian passport numbers. It ensures that the entered passport number follows the correct format and provides real-time feedback to users.

## Usage

```tsx
import { IndianPassport } from "@workspace/domain-ui-registry/components/domain-ui/indian-passport";

function PassportForm() {
  const [passport, setPassport] = useState("");

  return (
    <IndianPassport
      value={passport}
      onChange={(e) => setPassport(e.target.value)}
      placeholder="Enter passport number"
    />
  );
}
```

## Validation Rules

Indian passport numbers follow this format:

- **First Character**: Letter (A-Z)
- **Next 7 Characters**: Digits (0-9)
- **Total Length**: 8 characters

Example: `A1234567`
