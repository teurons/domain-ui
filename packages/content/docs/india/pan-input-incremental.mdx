## Features

The PAN Input Incremental component provides real-time validation for Indian PAN numbers with:

- **Character-by-character validation** - Ensures each character follows PAN format rules
- **Invalid character prevention** - Blocks characters that don't match PAN pattern
- **Visual feedback** - Shows validation state with color-coded borders
- **PAN format enforcement**: `ABCDE1234F`
  - First 3 characters: Letters (A-Z)
  - 4th character: Person type (P, C, F, T, A, B, G, H, J, L, E)
  - 5th character: First letter of surname (A-Z)
  - Next 4 characters: Numbers (0-9)
  - Last character: Check letter (A-Z)

## Usage

```tsx
import { PanInputIncremental } from "@workspace/domain-ui-registry/components/domain-ui/india/pan-input-incremental";

export function PanForm() {
  return (
    <PanInputIncremental
      placeholder="Enter PAN number"
      onValidation={(status) => {
        if (status === "valid") {
          console.log("Valid PAN entered");
        }
      }}
    />
  );
}
```

## Validation States

- **Green border**: Valid complete PAN number
- **Amber border**: Incomplete but valid so far
- **Red border**: Invalid character entered
